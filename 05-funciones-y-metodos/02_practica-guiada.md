# Practica guiada — Funciones y metodos

## Objetivo

Separar responsabilidades y reutilizar logica.

## Actividades

1. Crea un metodo `Sumar(int a, int b)`.
2. Crea un metodo `EsPar(int numero)` que retorne `bool`.
3. Crea un metodo con parametro opcional para saludar.
4. Usa una lambda para multiplicar dos numeros.
5. Crea un metodo que reciba una lista y muestre sus elementos.

## Mini reto

Organiza el programa como una mini calculadora con metodos separados para suma, resta, multiplicacion y division.

## Como validar

```bash
cd 05-funciones-y-metodos
dotnet run
```
