# Practica guiada — Manejo de errores

## Objetivo

Construir programas mas seguros y faciles de depurar.

## Actividades

1. Fuerza un error de division entre cero y capturalo.
2. Usa `try/catch/finally` para registrar el flujo.
3. Valida una entrada numerica con `int.TryParse`.
4. Crea una excepcion personalizada simple.
5. Usa `using` para manejar recursos.

## Mini reto

Simula un cajero o calculadora que nunca se caiga aunque el usuario ingrese datos invalidos.

## Como validar

```bash
cd 10-manejo-de-errores
dotnet run
```
