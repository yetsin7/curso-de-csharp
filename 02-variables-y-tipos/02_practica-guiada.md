# Practica guiada — Variables y tipos

## Objetivo

Entender como C# representa datos y como elegir el tipo correcto.

## Actividades

1. Declara una variable `decimal precioCurso = 19.99m`.
2. Declara una variable nullable como `int? edadOpcional = null`.
3. Convierte una cadena `"25"` a entero con `int.Parse` o `Convert.ToInt32`.
4. Usa interpolacion para mostrar un resumen del estudiante.
5. Declara una constante con `const string Pais = "Nicaragua";`.

## Mini reto

Crea un bloque de salida que represente una ficha de alumno con nombre, edad, nota y si aprobo.

## Como validar

```bash
cd 02-variables-y-tipos
dotnet run
```
