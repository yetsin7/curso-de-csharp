# Practica guiada — Proyecto Biblia

## Objetivo

Aplicar C#, SQL y acceso a datos en un proyecto util y real.

## Actividades

1. Ejecuta la consulta de los primeros 10 libros.
2. Cambia la busqueda a otro versiculo conocido.
3. Busca una palabra clave distinta como `fe`, `gracia` o `paz`.
4. Ajusta el limite del top de libros con mas versiculos.
5. Agrega una consulta nueva para contar libros por testamento.

## Mini reto

Crea una funcion que reciba el nombre de un libro y muestre los primeros 5 versiculos del capitulo 1.

## Como validar

```bash
cd 13-proyecto-biblia
dotnet run
```
