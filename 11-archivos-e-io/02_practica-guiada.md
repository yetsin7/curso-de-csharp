# Practica guiada — Archivos e I/O

## Objetivo

Leer, escribir y organizar datos en disco con seguridad.

## Actividades

1. Crea un archivo de texto con `File.WriteAllText`.
2. Lee su contenido con `File.ReadAllText`.
3. Crea una carpeta de pruebas con `Directory.CreateDirectory`.
4. Lista archivos de una carpeta.
5. Serializa un objeto sencillo a JSON.

## Mini reto

Construye un mini registro de tareas que guarde y lea informacion desde disco.

## Como validar

```bash
cd 11-archivos-e-io
dotnet run
```
