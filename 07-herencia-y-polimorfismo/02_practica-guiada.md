# Practica guiada — Herencia y polimorfismo

## Objetivo

Reutilizar codigo y cambiar comportamientos sin duplicar logica.

## Actividades

1. Crea una clase base `Empleado`.
2. Crea dos clases derivadas: `Desarrollador` y `Disenador`.
3. Sobrescribe un metodo `DescribirRol()`.
4. Guarda instancias en una lista del tipo base.
5. Recorre la lista y ejecuta el mismo metodo para ver el polimorfismo.

## Mini reto

Modela animales con un metodo `HacerSonido()` y demuestra como cada clase responde distinto.

## Como validar

```bash
cd 07-herencia-y-polimorfismo
dotnet run
```
