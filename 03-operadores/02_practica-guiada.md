# Practica guiada — Operadores

## Objetivo

Practicar calculos, comparaciones y expresiones utiles en programas reales.

## Actividades

1. Calcula subtotal, impuesto y total de una compra.
2. Evalua si un usuario es mayor de edad con `>=`.
3. Usa `&&` y `||` para validar acceso.
4. Usa el operador ternario para mostrar `"Aprobado"` o `"Reprobado"`.
5. Prueba `??` para asignar un valor por defecto cuando una variable sea `null`.

## Mini reto

Simula el calculo de descuento de una tienda: si el total supera cierto monto, aplica rebaja y muestra el total final.

## Como validar

```bash
cd 03-operadores
dotnet run
```
