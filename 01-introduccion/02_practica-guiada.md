# Practica guiada — Introduccion

## Objetivo

Familiarizarte con la consola, la salida de texto y la estructura minima de un programa en C#.

## Actividades

1. Cambia el saludo principal por uno con tu nombre.
2. Agrega una variable `string ciudad` y muestrala en consola.
3. Agrega una variable `bool estudiaCSharp` e imprime su valor.
4. Modifica `Saludo.Mostrar("Programador")` por dos nombres distintos.
5. Muestra la fecha actual con `DateTime.Now`.

## Mini reto

Crea una salida final que muestre:

- nombre;
- ciudad;
- edad;
- version de .NET;
- directorio actual.

## Como validar

```bash
cd 01-introduccion
dotnet run
```
