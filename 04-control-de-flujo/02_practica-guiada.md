# Practica guiada — Control de flujo

## Objetivo

Tomar decisiones y repetir instrucciones de forma controlada.

## Actividades

1. Crea un `if/else` para clasificar una nota.
2. Usa `switch` para traducir un numero de dia a su nombre.
3. Recorre una lista de nombres con `foreach`.
4. Usa `for` para mostrar la tabla del 5.
5. Usa `while` para contar hasta 10.

## Mini reto

Construye un menu simple de consola que muestre tres opciones y responda distinto segun la opcion elegida.

## Como validar

```bash
cd 04-control-de-flujo
dotnet run
```
