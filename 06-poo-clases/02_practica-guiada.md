# Practica guiada — POO, clases y objetos

## Objetivo

Modelar entidades del mundo real con propiedades, constructores y metodos.

## Actividades

1. Crea una clase `Curso` con nombre y precio.
2. Agrega un metodo `MostrarResumen()`.
3. Crea dos instancias distintas y comparalas visualmente.
4. Agrega validacion basica a una propiedad.
5. Crea un `record` para representar una categoria.

## Mini reto

Modela un producto digital con nombre, precio, autor y metodo para aplicar descuento.

## Como validar

```bash
cd 06-poo-clases
dotnet run
```
