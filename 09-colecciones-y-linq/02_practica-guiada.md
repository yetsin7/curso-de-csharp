# Practica guiada — Colecciones y LINQ

## Objetivo

Trabajar con grupos de datos y hacer consultas expresivas.

## Actividades

1. Crea una `List<string>` con nombres de cursos.
2. Crea un `Dictionary<string, decimal>` con precios.
3. Filtra elementos con `Where`.
4. Ordena resultados con `OrderBy`.
5. Proyecta datos con `Select`.

## Mini reto

Crea una lista de productos y obtén:

- solo los activos;
- los ordenados por precio;
- el promedio de precio;
- el mas caro.

## Como validar

```bash
cd 09-colecciones-y-linq
dotnet run
```
